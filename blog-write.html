✏️ 블로그 글 작성

제목·내용을 작성한 뒤 게시 파일 저장 → PC에서 명령 실행 → GitHub push 하면 사이트에 올라갑니다.

미리보기

왼쪽에서 작성 후 「미리보기」를 누르세요.

📤 게시 방법 (3단계)

  1. 위에서 글 작성 → 게시 파일 저장 클릭 → blog-draft.json 다운로드 (다운로드 폴더에 저장돼도 됩니다)
  2. PowerShell에서 프로젝트 폴더로 이동 후 실행: 
    cd "d:\경락\커서\site setting\300site\10start"
python scripts/publish_blog.py --add blog-draft.json
    ※ 파일이 다운로드 폴더에만 있어도 자동으로 찾습니다.
  3. GitHub에 올리기: 
    git add data/blog-posts.json blog.html blog/ css/common.css css/blog.css sitemap.xml
git commit -m "블로그 글 추가"
git push origin main
